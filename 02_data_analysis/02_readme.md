## データ分析プロジェクト
ここではRを使ったデータ分析の模擬プロジェクトの説明と分析結果を掲載しています。

## Rとは？
Rは統計解析やデータ分析に特化した無料のソフトウェアです。<br>
構文がとてもシンプルで、散布図やグラフを簡単に作成することができます。
